====================================================
Weapons of Destruction 5.4 Commands quick reference
I recommend that you print this out.
====================================================
-
Super Blaster: Press 1 twice
Sniper Gun: Press 2 twice when you have the shotgun
Homing Rocket Launcher: Press 7 twice when you have the rocket launcher
Plasma Gun: Press 8 twice when you have the hyperblaster
Freeze Gun: Select Super Shotgun twice
Select the Machine Gun again for standard.
Select Flamethrower[9] twice for the standard railgun

To use darKMajick grenades (yes the spelling is intentional):
a.) Select the grenade from the heads up display by using [ and ],
-OR-
bind a key to "use grenades" and press the key several times
b.) Press enter again after selecting the grenades, it will go through
a list and text up at the screen of which grenade you have selected.

* Tip * Use console command "give all" to get all the items.
